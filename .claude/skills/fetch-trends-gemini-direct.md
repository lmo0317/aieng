# Skill: Fetch Trends Gemini Direct

이 스킬은 외부 LLM API(Gemini, GLM 등)를 사용하지 않고, Gemini CLI(나)의 자체 지능을 활용하여 실시간 뉴스 트렌드를 수집하고 최고 품질의 학습 데이터를 생성하여 로컬 DB에 저장하는 가이드입니다.

## 1. 목적
- 외부 API 호출 비용(Token 비용)을 0으로 절감.
- 트렌드 10개 × 학습 문장 10개 = 총 100개의 데이터를 누락 없이 생성.
- 대치동 영어 일타 강사 수준의 고퀄리티 해설 제공.

## 2. 수행 프로세스

### 단계 1: 실시간 뉴스 제목 수집 (반드시 10개)
- `node fetch-news-titles.js`를 실행하여 최신 뉴스 제목 10개를 가져옵니다. (결과가 깨질 경우 파일로 저장 후 `read_file`로 읽어옵니다.)
- **[주의]** 임의로 개수를 줄이지 말고, 반드시 수집된 10개의 트렌드 전체를 다음 단계의 대상으로 삼아야 합니다.
- (필요시 다음 RSS 주소 활용: `https://news.google.com/rss?hl=ko&gl=KR&ceid=KR:ko` 등)

### 단계 2: 학습 콘텐츠 생성 (Gemini 직접 수행, 10개 트렌드 모두 처리)
수집된 10개의 뉴스 제목 각각에 대해 예외 없이 모두 아래 데이터를 생성합니다. 당신은 트렌드 맞춤형 영어 학습 서비스 'Trend Eng'의 전문 AI 튜터입니다.

- **category**: 뉴스 분야 (연애, 테크, 전체, 스포츠 등)
- **title**: 뉴스 제목
- **summary**: 한국어 요약 (1~2문장)
- **keywords**: 핵심 키워드 3개 배열
- **difficulty**: `level3` 고정
- **sentences**: **[가장 중요] 무조건 10개의 문장 배열 (절대 5개 등 임의로 줄이지 마세요)**. 각 문장은 다음 5가지 요구 조건을 완벽히 충족해야 합니다.
  
  1. `en`: 주제 기반으로 선택한 레벨(level3: 중급)에 맞는 난이도의 영어 문장.
  2. `ko`: 해당 영어 문장에 대한 자연스러운 한국어 해석.
  3. `sentence_structure`: 문장의 형식(1~5형식)과 주요 문장 성분(주어, 동사, 목적어, 보어, 수식어 등)을 상세히 분석.
     - *예시: `5형식 / 주어: The huge price gap, 동사: makes, 목적어: the choice, 목적격 보어: difficult`*
  4. `explanation`: **[중요: 대치동 영어 일타 강사 모드]** "자, 여러분 집중! 여기서 끊어 읽으세요!", "이 구문은 시험에 무조건 나옵니다!" 와 같이 과외 선생님이 옆에서 알려주듯 생동감 넘치고 친절한 말투로 작성하세요. 단순 뜻풀이를 넘어, 직독직해 요령, 원어민의 뉘앙스, 문법적 원리를 3~4문장으로 길고 아주 상세하게 설명해야 합니다. 절대 한 줄로 대충 넘기지 마세요.
  5. `voca`: 문장에 쓰인 핵심 단어와 숙어 표현 정리. (예: `["compare A with B: A와 B를 비교하다", "budget-friendly: 가성비가 좋은"]`)

**[출력 토큰 제한 극복 노하우]**
- 100개의 문장을 한 번에 생성하다가 글자 수 제한으로 문장이 잘리거나 5개로 줄어드는 현상을 방지하기 위해, 데이터 생성을 Part 1(1~5번 트렌드)과 Part 2(6~10번 트렌드)로 나누어 각각의 `.js` 파일에 배열 형태로 저장해도 좋습니다.

### 단계 3: DB 저장 (단일 API 호출)
- **[매우 중요]** 서버의 `/api/trends/save` 엔드포인트는 호출될 때마다 해당 날짜의 기존 데이터를 모두 지우고 덮어씌우는 로직을 가지고 있습니다.
- 따라서 Part 1, Part 2로 나누어 데이터를 생성했다 하더라도, **반드시 두 배열을 하나로 합친 뒤(총 10개 트렌드) 서버로 단 한 번만 POST 요청을 보내야 합니다.**
- 전송 예시 코드:
  ```javascript
  const axios = require('axios');
  const part1 = require('./part1.js');
  const part2 = require('./part2.js');
  const allTrends = [...part1, ...part2];
  
  axios.post('http://localhost:80/api/trends/save', { trends: allTrends })
      .then(res => console.log('성공:', res.data))
      .catch(err => console.error('에러:', err.message));
  ```
- 포트는 상황에 따라 `80` 또는 `3000`을 확인하여 사용합니다. (`netstat -ano | findstr :80` 등 활용)

## 3. 실행 명령어 예시
사용자가 다음과 같이 요청할 때 이 스킬을 활성화합니다:
- "트렌드 뉴스 수집 해줘" (가장 명시적인 호출)
- "비용 없이 뉴스 트렌드 업데이트해줘"
- "Gemini 직접 모드로 뉴스 수집 시작해"
