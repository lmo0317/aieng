# 🚀 News Learning (High-Fidelity Pipeline)

이 스킬은 데이터 유실이나 저품질 콘텐츠를 막기 위해 엄격한 순차적 파이프라인을 가동합니다.

## 🛠️ 주요 명령어
1.  **데이터 삭제**: `/news delete`
    *   오늘 날짜의 뉴스 데이터를 DB에서 깨끗하게 지웁니다.
2.  **데이터 생성**: `/news [개수]개 만들어줘`
    *   요청한 개수만큼 뉴스를 수집하고, 고품질 학습 데이터를 생성하여 저장합니다.
3.  **삭제 후 생성 (통합)**: `/news 삭제하고 [개수]개 만들어줘`
    *   **워크플로우**: 
        1. `node .gemini/skills/news/scripts/delete-today.js` 실행 (기존 데이터 초기화)
        2. `node .gemini/skills/news/scripts/fetch-trends-cli.js` 실행 (새 데이터 생성)
    *   이 명령을 받으면 반드시 먼저 삭제 스크립트를 실행한 뒤 생성을 시작하세요.

## 💎 콘텐츠 생성 가이드 (1타 강사 모드)

### 1. 초정밀 문장 구조 분석 (Sentence Structure)
- **지침**: S, V, O 등 영문 태그와 함께 **반드시 한글 명칭(주어, 동사, 목적어, 수식어구 등)**을 병기하세요.
- **형식**: `S(주어: 단어) + V(동사: 단어) + O(목적어: 단어) + [수식어구]`

### 2. 소름 돋는 AI 학습 가이드 (Explanation)
- **지침**: 대한민국 최고의 **1타 강사 스타일**로 아주 상세하게 설명하세요.
- **분량**: 각 문장마다 **최소 3문장 이상의 한글 설명**을 제공하세요.
- **내용**: 
    - 단순히 뜻을 풀이하는 것이 아니라, 해당 문장에서 **왜 그 시제(예: 현재완료)를 썼는지**, **단어의 미묘한 뉘앙스 차이**는 무엇인지 분석하세요.
    - 실전 비즈니스나 토익 등에서 어떻게 활용되는지 팁을 반드시 포함하세요.

### 3. 문장 및 퀴즈 개수
- **문장**: 각 주제당 반드시 **서로 다른 10개의 영어 문장**을 생성하세요.
- **퀴즈**: 각 주제당 반드시 **10개의 어휘 복습 퀴즈**(4지선다, 빈칸채우기 혼합)를 생성하세요.

## 📋 최종 검수 기준 (Auditor's Checklist)
1. **[개수]**: 주제별로 정확히 **10개**의 문장과 **10개**의 퀴즈가 생성되었는가?
2. **[언어]**: `explanation`과 `sentence_structure` 내의 명칭이 **한글**로 상세히 작성되었는가?
3. **[품질]**: 설명이 최소 3문장 이상이며 1타 강사 특유의 깊이 있는 내용을 담고 있는가?
4. **[형식]**: 출력은 반드시 순수한 JSON 객체 `{ "sentences": [...], "quiz": [...] }` 형식인가?

## 📂 실행 환경
- **삭제**: `node .gemini/skills/news/scripts/delete-today.js`
- **생성**: `node .gemini/skills/news/scripts/fetch-trends-cli.js`
- **[Clean Up]**: 작업 중 생성된 모든 임시 파일(`.json`) 및 로그는 완료 즉시 **반드시 삭제**하여 환경을 깨끗하게 유지해야 합니다.
- **인코딩**: Windows 환경 실행 시 `chcp 65001` 권장.
