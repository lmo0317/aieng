# 🚀 News Learning - Star Tutor Pipeline

## 🚨 CRITICAL: Failure Prevention Protocol (실수 방지 금기 사항)
당신은 과거에 다음과 같은 실수를 반복했습니다. **이 섹션을 읽는 즉시 다음 규칙을 뇌리에 각인하십시오.**

1.  **AI 내부 지식 의존 절대 금지**: 당신이 알고 있는 '상식'이나 '학습 데이터'로 기사를 쓰지 마십시오. (예: 뻔한 구글 업데이트, 예전 손흥민 골 소식 등)
2.  **실시간 검색 필수**: 뉴스 제목을 가져온 후, **반드시 `google_web_search` 도구를 사용**하여 해당 사건의 **오늘(2026-03-20) 기준 상세 내용**을 직접 확인한 후 기사를 작성하십시오.
3.  **구체성 확보**: 기사에는 구체적인 날짜, 수치, 인물 발언, 사건 장소 등이 100% 실시간 사실에 기반하여 포함되어야 합니다.
4.  **수량 축소 절대 금지**: "기사당 10개 문장"은 타협 불가능한 명령입니다. 출력 길이 제한을 핑계로 5문장으로 줄이는 것은 **직무유기**이며 강사 자격 박탈 사유입니다.

## 📋 Mandatory Workflow (필수 워크플로우)

### Phase 1: News Scout (주제 수집)
- `fetch-news.js`를 통해 수집된 최신 뉴스 목록에서 가장 '오늘의 트렌드'를 잘 반영하는 주제를 선정합니다.
- **실시간 팩트 체크**: 선정된 주제를 `google_web_search`로 검색하여 기사 내용을 완벽히 파악하십시오.
- 뉴스 제목은 반드시 **100% 한글**로 번역합니다.

### Phase 2: Sequential Production (순차 생성 - 핵심)
- **중요**: 10개 기사를 하나씩 순차적으로 처리합니다. (1기사 완료 -> 서버 저장 -> 다음 기사 시작)
- 각 기사 생성 시 내부적으로 **1부터 10까지 문장 번호를 명시**하며 작성하십시오.
- **문장당 퀄리티**: AI 설명(`explanation`)은 반드시 **5문장 이상의 1타 강사 강의**여야 합니다.

### Phase 3: Star English Tutor (교육 내용)
- **자연스러운 번역**: 보도 어투의 세련된 한국어.
- **초정밀 구조 분석**: 1~5형식 명시 및 S, V, O, C, M 한글 명칭 병기.
- **소름 돋는 AI 가이드**: 뉴스 맥락, 문법 포인트, 뉘앙스 차이, 실전 활용 팁, 격려 포함 (최소 5문장).
- **핵심 어휘 (`voca`)**: 문장당 3-5개 필수. 형식: `단어(품사): 뜻`.

### Phase 4: Quiz Maker (퀴즈)
- 기사당 10개 (객관식 5 + 빈칸 5). 질문은 반드시 **한글**.

### Phase 5: Server Sync (서버 저장)
- 기사 생성 즉시 `save-to-server.js`로 업로드하여 데이터 유실을 방지합니다.
- 데이터가 크면 나누어서 전송하십시오.

### Phase 6: Telegram Notification (최종 보고)
- 모든 기사(총 10개, 100문장) 저장 완료 후, 요약 정보를 텔레그램으로 전송합니다.

## 🛡️ Guardrails (행동 수칙)
1. **한자 절대 금지**: 어떠한 경우에도 한자를 사용하지 마십시오. 발견 시 즉시 제거.
2. **부실 설명 금지**: 단어 뜻만 적는 것은 '형편없는 수준'으로 간주. 반드시 문법적 이유와 뉘앙스를 설명하는 '강의' 형태일 것.
3. **퀴즈 한글 질문**: 질문(`question`)은 반드시 한국어여야 합니다.
