# 🎵 Pop Song Learning (1타 강사 Edition)

이 스킬은 팝송 가사 전문을 분석하여, 실생활 표현과 문법을 동시에 잡는 5단계 하이엔드 워크플로우를 가동합니다.

## 🛠️ 권장 입력 형식
*   **기본 (전체 가사)**: `/popsong 가수 - 노래제목`
*   **부분 요청**: `/popsong 가수 - 노래제목 (5문장만 해줘)`

## 💎 콘텐츠 생성 가이드 (1타 강사 모드)

### 1. 가사 스카우터 & 에디터
- **가사 확보**: 노래 제목과 가수를 바탕으로 **가사 전문(Full Lyrics)**을 확보하세요.
- **이미지**: 해당 곡의 분위기를 잘 나타내는 앨범 아트나 관련 이미지 URL을 찾아 `image` 필드에 포함하세요.
- **편집**: 중복되는 후렴구(Chorus)는 학습 효율을 위해 적절히 생략하거나 한 번만 처리하세요.

### 2. 초정밀 문장 구조 분석 (Sentence Structure) - CRITICAL
- **지침**: S, V, O 등 영문 태그와 함께 **반드시 한글 명칭(주어, 동사, 목적어, 수식어구 등)**을 병기하여 직관적으로 분석하세요.
- **필수 형식**: `S(주어: 영단어) + V(동사: 영단어) + O(목적어: 영단어) + [수식어구]`
- **예시**: `S(주어: I) + V(동사: think) + O(목적어: I wanna marry you)` 와 같은 형태로 한글 명칭이 빠지지 않게 하세요.

### 3. 소름 돋는 AI 학습 가이드 (Explanation)
- **지침**: 대한민국 최고의 **1타 강사 스타일**로 아주 상세하게 설명하세요.
- **분량**: 각 문장마다 **최소 3~4문장 이상의 한글 설명**을 제공하세요.
- **내용**: 
    - 가사 특유의 **뉘앙스, 슬랭, 관용구, 비유적 표현**을 중심으로 설명하세요.
    - 실제 원어민들이 이 표현을 어떤 상황에서 쓰는지 실전 팁을 포함하세요.

### 4. 문장 및 퀴즈 개수
- **문장**: **가사 전문(Full Lyrics)의 모든 문장을 한 줄도 누락하지 말고** 학습 데이터로 생성하세요. 5개나 10개로 요약하는 행위를 절대 금지하며, 노래 전체를 공부할 수 있도록 모든 텍스트를 포함해야 합니다. (단, 완전히 동일한 가사가 반복되는 코러스는 한 번만 처리 가능)
- **퀴즈**: 각 노래당 반드시 **10개의 어휘 복습 퀴즈**(4지선다, 빈칸 채우기(일부 철자 가림))를 생성하세요.

## 📋 최종 검수 기준 (Auditor's Checklist)
1. **[누락 엄금]**: **가사 전문의 모든 문장**이 포함되었는가? (일부만 발췌하거나 요약했다면 반려)
2. **[언어]**: `explanation`과 `sentence_structure`가 **한글**로 상세히 작성되었는가?
3. **[퀄리티]**: 가사의 감성과 실전 영어를 모두 담은 1타 강사 스타일인가?
4. **[형식]**: 출력은 반드시 지정된 JSON 구조를 유지하는가?

## 📂 실행 및 서버 동기화
1.  생성된 JSON 데이터를 `output/` 폴더에 임시로 저장합니다.
2.  `node .gemini/skills/popsong/scripts/save-song.js <파일경로>` 명령어로 서버에 업로드합니다.
3.  **[Clean Up - 필수]**: 업로드가 성공하면 생성된 JSON 파일 및 기타 모든 임시 작업 파일을 **즉시 삭제**하여 프로젝트 루트를 깨끗하게 유지하세요.
